# 🎵 Discord Music Bot

A simple Discord music bot built with discord.js v14.

## 🚀 Features
- Play music from YouTube
- Queue system
- Skip / Stop commands
- 24/7 hosting support (Railway)

## ⚙️ Setup

### 1. Install dependencies
npm install

### 2. Add environment variables
TOKEN=your_bot_token
PREFIX=!

### 3. Run bot
npm start

## 🎮 Commands
!play song name
!skip
!stop
